## 🧠 What is a Data Science Engineer?

A Data Science Engineer works with data to:

* Collect and clean data
* Analyze patterns
* Build models
* Create data-driven solutions

---

## 🛠️ Skills You Need

### 1. Programming Languages

* Python (Most important)
* SQL
* Optional: R

### 2. Mathematics & Statistics

* Probability
* Linear Algebra (basic)
* Statistics (mean, median, variance)

### 3. Data Handling

* Data cleaning
* Data transformation
* Working with CSV, Excel, JSON

### 4. Data Visualization

* Matplotlib
* Seaborn
* Power BI / Tableau

### 5. Machine Learning (Basics)

* Supervised Learning
* Unsupervised Learning
* Model evaluation

## 📊 Example Code (Python)

```python
import pandas as pd

# Load dataset
data = pd.read_csv("data.csv")

# Show first 5 rows
print(data.head())

# Check missing values
print(data.isnull().sum())

# Basic statistics
print(data.describe())
```
